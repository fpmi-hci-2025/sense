## Профили задач

| Задача | Действия | Контекст | Критерий выполнения |
|--------|----------|----------|---------------------|
| найти нужную информацию | ввести запрос в поиск или выбрать категорию | использование веб-приложения или мобильного устройства | пользователь получил список релевантных материалов |
| опубликовать контент (пост, статью, фото/видео) | нажать кнопку «Создать», ввести текст, прикрепить медиа и опубликовать | спокойная обстановка, стабильное интернет-соединение | контент успешно опубликован и доступен другим пользователям |
| прокомментировать публикацию | перейти к посту, ввести текст комментария, нажать «Отправить» | произвольный | комментарий появился под публикацией |
| зарегистрироваться в системе | заполнить форму регистрации, подтвердить email/телефон | первое использование, доступ к интернету | пользователь успешно вошёл в систему |
| прочитать статью или пост | выбрать публикацию из ленты или поиска | произвольный | текст отобразился на экране, пользователь может просмотреть весь материал |
| поделиться публикацией | нажать кнопку «Поделиться», выбрать способ (ссылка, соцсети) | при просмотре материала | публикация доступна другим людям через выбранный канал |
| поставить «лайк» публикации | нажать кнопку «Лайк» под постом или комментарием | произвольный | отметка отображается, счётчик увеличился |
| сохранить публикацию | нажать кнопку «Сохранить» или «В закладки» | при просмотре материала | публикация появилась в личных закладках |
| подписаться на автора | нажать кнопку «Подписаться» на странице автора | понравился контент при чтении, пользователь является знакомым | автор добавлен в список подписок, пользователь получает его обновления |

## Профили среды
| Характеристика | Признаки (для выбранной группы) | Влияние на интерфейс / UX |
|----------------|--------------------------------|---------------------------|
| **Место использования** | Дом; учеба/работа; транспорт (метро/автобус); улица/кафе; перемещения между локациями | Обязательный офлайн/нестабильный режим (кэш, очередь отправки постов); быстрый доступ к «Создать заметку»; крупные интерактивные области (≥44–48dp); минимизация шагов до публикации (≤3 шага); адаптивная верстка под разные плотности/оригинации |
| **Рабочее место** | Стоя/на ходу; одной рукой; телефон в чехле/перчатках; короткие сессии; иногда десктоп | One-hand UI (нижняя навигация, жесты), поддержка «быстрых действий»; клавиатура/голосовой ввод с автосохранением; крупные цели для большого пальца; фиксированная нижняя панель; упрощённые формы; поддержка drag&drop медиа на десктопе |
| **Освещённость** | Яркое солнце на улице; низкая освещённость вечером; мерцающее/неравномерное освещение | Высокая контрастность и достаточный размер шрифтов; темная/светлая тема с авто-переключением; четкие иконки/контуры; ограничение «тонких» цветов; индикация состояния не только цветом (форма/текст) |
| **Аппаратное обеспечение** | Смартфоны 5–6.7″; вырезы/«чёлки»; 2–8 ГБ RAM; различная мощность CPU/GPU; ограниченное хранилище; камеры как ключевой сценарий | Оптимизация веса медиа (сжатие, адаптивные битрейты); ленивая загрузка и скелетоны; плавность на слабых девайсах (минимум тяжелых анимаций); учет safe-area; контроль места (очистка кеша, предупреждения); быстрый старт камеры/галереи |
| **Программное обеспечение (ОС, браузер)** | iOS/Android (разные версии); системные браузеры (Chrome/Safari) и WebView/PWA; языки/локали; системный режим «Экономия энергии» | Соответствие гайдлайнам платформ (Share Sheet, «Войти с Apple/Google»); фоллбеки для старых версий; PWA-кэш и сервис-воркеры (если веб); корректная локализация и RTL-готовность; уважение системных настроек шрифта/темы; «чистые» запросы разрешений (камера/микрофон/уведомления) |
| **Прерывания** | Входящие звонки/сообщения; потеря сети; переключение приложений; разряд батареи; ограничение фоновой активности | Автосохранение черновиков; идемпотентная публикация и повтор отправки при восстановлении сети; сохранение состояния экрана; пауза/возобновление видео; неблокирующие ошибки; напоминание «Продолжить пост» после возврата |
| **Шумность** | Тихо (дом/офис) ↔ шумно (улица/транспорт); использование наушников; публичные места | Медиа по умолчанию без звука, клип с автосубтитрами; четкая визуальная обратная связь (вибро/хаптика, прогресс-индикаторы); управляемые звуки уведомлений; фильтр шумоподавления при записи; крупные визуальные подсказки вместо звуковых |
